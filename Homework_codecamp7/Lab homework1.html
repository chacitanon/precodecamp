2. ให้หาความเหมือนและความแตกต่างระหว่าง 2 tag ที่กำหนดให้

Name_Tag Similarity Difference
hr & br
    give line break.
    hr give a horizontal link break but br gives an empty line break
ul & ol
    define list.
    ul defines an unordered (bulleted) list but.ol defines an ordered list.
select & checkbox & radio button
    user to choose something.
    select used to create a drop-down list.
    checkbox is shown as a square box that is ticked when activaterd
    radio button in a group can be selected at the same time.
div & span
    define section or division in HTML.
    div defines a division or a section in an HTML document. but span is an inline container used to mark up a part of a text, or a part of a document.
script & style
    use external documents.
    style information (CSS) for a document but script client-side script (JavaScript).
